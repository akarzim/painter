# Painter

## How to install?

Install the dependencies.

```
gem install bundler
bundle install
```

## How to run locally?

Run the middleman server.

```
bundle exec middleman
```

## How to build?

Build the sources and minimize the assets.

```
bundle exec middleman build
```

## How to deploy?

Publish on GitHub Pages.

```
bundle exec middleman gh_pages
```
